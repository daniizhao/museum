## How to execute

1. Open CLIPS IDE
2. Go to File -> Load... (Ctrl+L) and select "museum.clp" of this repository.
3. Go to Execution -> Reset (Ctrl+E) to instantiate the instances.
4. Go to Execution -> Run (Ctrl+R) and answer the questions!
